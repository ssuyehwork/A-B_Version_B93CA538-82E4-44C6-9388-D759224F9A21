# SearchTool - 高级搜索工具 (独立版)

这是从 RapidNotes 提取的独立搜索工具应用。

## 项目架构

本项目采用符合专业规范的目录结构：

- `src/`: 核心源代码
  - `windows/`: 窗口界面逻辑 (主窗口、搜索窗口、无边框基类)
  - `widgets/`: 自定义通用控件 (可点击输入框等)
  - `utils/`: 工具类 (字符串处理、图标渲染、SVG 资源)
- `resources/`: 资源文件
  - `qss/`: 暗黑模式样式表
  - `icons/`: 应用图标
  - `resources.qrc`: Qt 资源定义

## 功能特性

1. **文件查找**: 支持全盘/指定目录扫描，具备增量扫描、收藏夹拖拽（支持置顶）、右键定位文件等功能。
2. **关键字查找**: 支持批量文本搜索与替换，提供备份与一键撤销机制。
3. **极速体验**: 采用 `QtConcurrent` 异步处理，确保 UI 永不卡顿。
4. **专业 UI**: 适配无边框圆角设计，支持窗口置顶和位置记忆。

## 构建说明

本项目已子项目化，可以通过根目录的 `CMakeLists.txt` 统一构建，也可以单独进入 `SearchApp` 目录进行构建。

依赖项：
- Qt 6 (Core, Gui, Widgets, Concurrent, Svg)
- C++ 20 兼容编译器
